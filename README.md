# AWS Cloud Storage Architecture

This project demonstrates the design of a scalable and cost-efficient cloud storage architecture using Amazon S3.

## Architecture

User Upload
     |
     v
Amazon S3 Bucket
     |
Lifecycle Policy
     |
Archive to Glacier

## Features

- S3 object storage
- lifecycle management
- versioning enabled
- cost optimization using storage tiers

## AWS Services Used

- Amazon S3
- AWS IAM
- S3 Lifecycle Policies
- CloudWatch monitoring

## Goal

This project demonstrates how to design a resilient cloud storage solution for long-term data storage and backup.
